#
## Install node.js
##
check in your console what version of node you have.
##
run command: node -v

#
## Install typescript
##
npm install -g typescript
##
check in your console for what version of typescript you have. 
##
run command: tsc -v (3.8.1)

# 
### Open a new file and name it main.ts

#
### Compile to JS:
##
in terminal, run command: $ tsc main.ts or tsc main
(will set up it's own JS file.)
##
in terminal, run command: $ node main.js or node main
##
then run command: $ tsc main --watch (will watch for file changes)

### Once coding begins:
##
open second terminal and run $ node main to check for output.
